1) Nhập dữ liệu API

  • Cart và order cần JWT access token hợp lệ.
  • Return URL nên trùng với VNPAY_RETURN_URL đã cấu hình/đăng ký với VNPay; dùng URL lạ rất dễ bị báo Dữ liệu gửi sang không đúng định dạng.
  • Nếu muốn nhận callback VNPay về lại page này, hãy mở file qua HTTP (Live Server hoặc static server), không nên dùng file://.
  • Nếu back-end báo VNPay is not configured, cần set VNPAY_TMN_CODE và VNPAY_HASH_SECRET.
